What CatFACS Is

The Cat Facial Action Coding System (CatFACS) is a scientific observational tool for identifying and coding facial movements in domestic cats. The system is based on the facial anatomy of cats and has been adapted from the original FACS system used for humans created by Ekman and Friesen (1978).

The CatFACS manual details how to use the system and code the facial movements of cats objectively. The manual and certification are freely available (see below).

More information regarding the development of this FACS system can be found in:

Caeiro, Cátia C., Anne M. Burrows, and Bridget M. Waller. "Development and application of CatFACS: Are human cat adopters influenced by cat facial expressions?" Applied Animal Behaviour Science 189 (2017): 66–78.

Read the research article

What CatFACS Is Not

CatFACS is not an ethogram of facial expressions and does not make any inference about underlying emotion or context causing the movement. Instead, it is an objective coding scheme with no assumptions about what represents a facial expression in this species. It does not explicitly teach cat facial expressions.
